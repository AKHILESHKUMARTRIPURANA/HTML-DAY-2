HTML DAY 2 INTERVIEW QUETIONS

8. Difference between HTML tags and elements?

Ans:HTML elements are created using HTML tags. Tags are enclosed between < & > symbols. Every HTML element has its own set of HTML tags. An HTML element is the content that appears on the webpage, while an HTML tag is the code used to define that content and specify how it should be displayed.

9. Explain br tag?

Ans: br tag:It is a line break and also a self closing tag. It is a block element, all elements occuring after the br will be sent to the next line. Used for creating a division line. When the webpage is rendered in a web browser, any content that comes after the br tag will appear on the next line.

10. Explain how many different types of lists we have in HTML?

Ans:In lists we are having 3 types, they are: a.Ordered List, b.Unordered List, c.Description List.

a.Ordered List(ol):There are 5 types of ordered lists (1,I,i,A,a). Default value is 1.

b.Unordered List(ul):There are 4 types of unordered lists (disc,square,circle,none). Default value is disc.

c.Description List(dl):There are 2 types of description lists (Description Title(dt),Description Data(dd))

11. What are semantic tags?

Ans:HTML tags are specified into two groups.They are:a.Semantic Tags, b.NonSemantic Tags

a.Semantic Tags:Tags that are specific about their functionality.Ex:image tag,button tag, heading tags, anchor tag.

b.NonSemantic Tags:Tags that are generally meant to group/wrap other HTML elements. These tags don't display anything by themselves.Ex:form,section,nav,header,footer,div,span tags.
